Monitoring: is an act of collecting and analyzing data to determine the performance, health and availability of the applications and the resouces that depends on it.
Increase the uptime of the application by knowing the issues proactively and resolve them before they occur.

Azure monitor: Using azure monitor we can monitor

2. Operating System
3. Azure Resources
4. Azure Subscriptions
5. Custom sources
